

# Python Operators

Operators are symbols used to perform operations on variables and values. Python provides multiple operator categories for mathematical calculations, comparisons and logical processing.

## Coverage

Topics:

- Arithmetic Operators
- Comparison Operators
- Logical Operators
- Assignment Operators
- Membership Operators
- Identity Operators

## Operator Types

### Arithmetic Operators

Used for mathematical operations.

Examples:

```python
x = 10
y = 5

print(x + y)
print(x - y)
print(x * y)
print(x / y)
print(x % y)
```

Coverage:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Exponent (**)

### Comparison Operators

Used to compare values.

Examples:

```python
cpu = 70
memory = 80

print(cpu > memory)
print(cpu == memory)
```

Coverage:

- Equal (==)
- Not Equal (!=)
- Greater Than (>)
- Less Than (<)
- Greater Than Equal (>=)
- Less Than Equal (<=)

### Logical Operators

Used for condition processing.

Examples:

```python
is_api_up = True
is_database_up = True

print(is_api_up and is_database_up)
```

Coverage:

- and
- or
- not

### Assignment Operators

Used for assigning values.

Examples:

```python
request_count = 100
request_count += 10
```

Coverage:

- =
- +=
- -=
- *=
- /=

### Membership Operators

Used for collection checks.

Examples:

```python
services = ["api", "database", "cache"]

print("api" in services)
```

Coverage:

- in
- not in
