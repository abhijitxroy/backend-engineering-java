

# Python Collections

Collections are data structures used to store and manage multiple values efficiently. Python provides built-in collection types for handling data processing and backend application development.

## Collection Types

### List

Lists are ordered and mutable collections.

Example:

```python
services = ["api", "database", "cache"]

services.append("queue")
```

Coverage:

- Create List
- Add Element
- Remove Element
- Update Element
- List Iteration

### Tuple

Tuples are ordered and immutable collections.

Example:

```python
server = ("localhost", 5432)
```

Coverage:

- Create Tuple
- Tuple Access
- Immutable Data Storage

### Set

Sets store unique values.

Example:

```python
regions = {"us-east", "eu-west", "us-east"}
```

Coverage:

- Unique Values
- Add Element
- Remove Element
- Set Operations

### Dictionary

Dictionary stores key-value pairs.

Example:

```python
config = {
    "host": "localhost",
    "port": 5432
}
```

Coverage:

- Key Value Storage
- Update Data
- Access Values
- Iteration
