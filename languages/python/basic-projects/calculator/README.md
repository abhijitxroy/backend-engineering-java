# Python Calculator Project

Calculator is a beginner-friendly Python project used to perform mathematical operations through user input. This project helps understand functions, control flow, operators and input handling.

## Coverage

Features:

- Addition
- Subtraction
- Multiplication
- Division
- User Input Processing
- Conditional Logic
- Function Usage
- Error Validation

## Project Objective

Build a command-line calculator application capable of processing arithmetic operations.

Supported operations:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

## Concepts Used

Project concepts:

- Variables
- Functions
- Operators
- if else Conditions
- Input Handling
- Type Conversion

Example:

```python
number1 = float(
    input(
        "Enter Number 1: "
    )
)

number2 = float(
    input(
        "Enter Number 2: "
    )
)

operator = input(
    "Enter Operator: "
)
```

Operation example:

```python
if operator == "+":

    print(
        number1 + number2
    )
```
