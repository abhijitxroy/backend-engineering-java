# Python Quiz Game

Quiz Game is a command-line Python project where users answer questions and receive scores based on correct answers. This project helps understand functions, conditions, user input handling and score tracking.

## Coverage

Features:

- User Input Processing
- Function Usage
- Conditional Logic
- Score Tracking
- Question Validation
- Interactive Gameplay

## Project Objective

Build an interactive quiz application where:

- User starts the game
- Questions are displayed
- Answers are validated
- Score is calculated
- Final result is displayed

## Concepts Used

Project concepts:

- Variables
- Functions
- if else Conditions
- String Processing
- Input Handling
- Function Return Value

Game start example:

```python
play = input(
    "Do you want to play game? (Y/N) "
)
```

Question validation example:

```python
if a1.lower() == "pradosh c meter":

    print(
        "Correct"
    )

    score += 1

else:

    print(
        "Wrong"
    )
```
