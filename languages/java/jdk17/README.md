

# JDK 17

JDK 17 is a Long-Term Support (LTS) release and the modern enterprise standard for Java development.

It introduced several language improvements that make code:

- Cleaner
- More Readable
- Less Boilerplate
- Easier To Maintain
- Better For Domain Modeling
