

# JDK 11

JDK 11 is a Long-Term Support (LTS) release and one of the most widely adopted Java versions in enterprise applications.

It introduced several improvements focused on:

- Developer Productivity
- Modern HTTP Communication
- String Processing
- File Handling
- Better API Usability
